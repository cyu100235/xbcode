插件目录说明
 - api ------ 接口目录，主要与其他插件对接，非网络请求接口
 - app ------ 应用目录
 - app/queue ------ 队列任务目录
 - app/home/controller ------ 默认控制器目录
 - app/model ------ 模型目录
 - config ------ 配置目录
 - data ------ 数据目录
 - public ------ 公共目录
 - view ------ Vue组件视图
 - setting ------ 设置目录
